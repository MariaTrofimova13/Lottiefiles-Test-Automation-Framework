## Lottiefiles-Test-Automation-Framework

## О проекте
Это учебный проект по разработке автоматизированного тестового фреймворка для сайта [lottiefiles.com](https://lottiefiles.com).  
**Цель** — автоматизация тестирования пользовательского интерфейса (UI) и прикладного программного интерфейса API веб-сайта.

## Функционал
- **Форма логина:**
    - Проверка успешной авторизации с валидными данными.
    - Обнаружение ошибок при вводе пустых полей, неправильного формата email, неправильного пароля.
    - Проверка отключения кнопки входа при неполных данных.
- **Поиск по сайту:**
    - Ввод поискового запроса.
    - Проверка отображения результатов поиска и корректность вывода поисковых баннеров.
    - Проверка поисковой команды через командную строку для сравнения поведения сайта.
- **API тестирование:**
    - Отправка запросов на сервер для авторизации с различными наборами данных.
    - Валидация ответов сервера и правильности обработки успешных и ошибочных ситуаций.

## Технологии и инструменты
- **Java 24**
- **Maven** — система сборки.
- **Selenium WebDriver** — автоматизация пользовательского интерфейса.
- **REST Assured** — тестирование API.
- **JUnit** — единичное тестирование, организация тестовых сценариев.
- **Java Faker** — генерация случайных тестовых данных.
- **Log4j2** — логирование выполнения тестов с сохранением логов в файл.

## Структура проекта
- Тестовые классы: разделены по сценариям логина, поиска и API.
- Вспомогательные классы: для генерации данных, Singleton и организации взаимодействия с браузером.
- Страницы (Page Objects): для структурированного взаимодействия с элементами UI.
